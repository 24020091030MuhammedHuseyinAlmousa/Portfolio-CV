# Muhammed Hüseyin Almousa - CV / Portfolyo

Kişisel portfolyo ve CV sayfası.  
HTML, CSS ve JavaScript kullanılarak hazırlanmıştır.  
Amacı: yazılım becerilerimi, projelerimi ve iletişim bilgilerini sergilemek.

## Özellikler

- Responsive tasarım, modern stil ve animasyonlar  
- Hakkımda, Yetenekler, Projeler ve İletişim bölümleri  
- İletişim formu (EmailJS ile gönderim)  
- Portfolyo görüntüleme amaçlıdır, izinsiz kullanım yasaktır

## Teknolojiler

- HTML5, CSS3, JavaScript  
- Google Fonts (Poppins)  
- EmailJS entegrasyonu
